Setup

  1. In firebase-config.js, paste your Firebase Web App config and ensure databaseURL is set for Realtime Database.
  2. In Firebase console, create a Realtime Database (test mode for quick start). Add simple security rules as needed.
  3. Host this folder using GitHub Pages or any static hosting.
